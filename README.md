# LinkUp - 綜合性活動社群與票務電商平台

LinkUp 是一個功能完整的 Monorepo 專案，結合了活動發佈、票務販售、電子商務商城以及社群投稿功能。本專案採用現代化的技術棧開發，提供流暢的用戶體驗與穩健的後端服務。

---

## 🚀 專案架構與技術棧

本專案使用 `pnpm` 工作區 (Workspaces) 管理前端與後端：

### 前端 (Client)
* **框架**: Next.js 16.0.1 (App Router)
* **核心庫**: React 19.2.0
* **樣式**: Tailwind CSS 4
* **UI 組件**: Headless UI, Heroicons, React Icons
* **數據視覺化**: Chart.js
* **輪播組件**: Swiper

### 後端 (Server)
* **框架**: Express 5.1.0
* **ORM**: Prisma 6.19.0
* **資料庫**: MySQL
* **語言**: TypeScript 5.9.3
* **驗證與安全**: Zod, Bcrypt, JSON Web Token (JWT)

---

## ✨ 核心功能

### 1. 用戶與權限管理
* **多重角色**: 支援一般會員 (`MEMBER`)、組織主辦方 (`ORGANIZER`) 及系統管理員。
* **第三方登入**: 整合 Google, GitHub, Facebook, LINE, Apple 及原生 APP 登入。
* **權限控管**: 區分使用者帳號、主辦方專屬資料以及管理員後台。

### 2. 活動與票務系統
* **活動類型**: 支援線上 (`ONLINE`) 與線下 (`OFFLINE`) 活動。
* **票務管理**: 自定義票券種類、價格、數量與銷售時間。
* **報到機制**: 內建 QR Code 數據生成與報到 (`Check-in`) 狀態追蹤。

### 3. 商城與結帳系統
* **產品規格**: 支援商品規格 (`Variants`)、庫存管理與價格偏離設定。
* **購物車**: 整合票券與商品項目的購物車系統。
* **訂單流**: 完整的訂單狀態追蹤（待付款、已付款、已出貨、已完成、已取消）。

### 4. 社群與互動
* **用戶投稿**: 使用者可發佈文章，支援分類、標籤與圖片上傳。
* **評論系統**: 針對活動與文章提供星級評分與留言功能.
* **收藏功能**: 可收藏活動、文章或主辦方。

### 5. 系統通知與公告
* **多態通知**: 涵蓋系統、活動、交易、審核與公告通知。
* **公告系統**: 管理員可發佈帶有封面圖的系統公告，並支援顯示順序控制。

## 🛠️ 開發環境設置

### 環境需求
* Node.js v20+
* pnpm 10.15.0+
* MySQL 資料庫

### 安裝與啟動

1. **安裝依賴**:
   ```bash
   pnpm install
   ```

2. **後端設置**:
   進入 `server` 目錄，配置 `.env` 檔案並初始化資料庫：
   ```bash
   cd server
   # 執行資料庫遷移
   pnpm prisma generate
   # 執行資料初始化
   pnpm prisma:seed
   ```

3. **啟動開發服務**:
   * **後端**: `pnpm run dev` (於 `server` 目錄執行)
   * **前端**: `pnpm run dev` (於 `client` 目錄執行)
